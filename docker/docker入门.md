## 1 docker 简介

### 1.1 docker 是什么？

docker 是一个容器解决方案，属于 linux 容器的一层封装  
docker 提供简单的接口，用户可以方便地创建和使用容器，容器可以实现对应用进行版本管理、复制、分享、修改

### 1.2 docker 要解决什么问题？

1. 提供一次性的环境
2. 提供弹性的云服务
3. 组件微服务架构
