# 网络-抓包

time: 2019.01.18

这里总结一下 wireshark 抓包学习总结

> 通过360软件或者电脑管家软件管理直接下载安装

## 1 基本功能

1. 开始/停止 录制
2. 协议过滤
3. 详细协议分析

![wireshark](../images/wireshark.png)

> 说明：wireshark 能捕捉到各种协议，比如 http, http2, tcp, ip, arp, smtp 等，是学习协议分析的好地方

## 2 同浏览器对比

| 功能描述     | wireshark  |  浏览器  |
| -------- | :-----: | :----:   |
| 协议学习 | 推荐 | 不推荐 |
| http协议结构化 | 不推荐 | 推荐 |
| 协议数据拷贝 | 不推荐 | 推荐 |
| 协议范围 | 推荐 | 不推荐 |
